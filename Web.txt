Langkah-langkah membuat Websitenya
1. Buat Database
2. Instal Bootstrap + buat Index
3. Koneksi Database + buat tampilan Gejala + CRUD Gejala
4. Buat Halaman Penyakit + CRUD Penyakit
5. Halaman Rules Base (Tampilan + Tambah data + simpan + update + hapus)
6. Halaman konsultasi

Proses Alur forward-chaining
A. dari gejala yang dipilih kemudian dicek ke rules base
B. Pilih gejala di halaman konsultasi

7. Halaman pembuatan users (pengguna)
   pasien, admin, dokter
   pasien = konsultasi

8. Halaman Login, hak akses
9. konsultasi buat admin dan dokter (role)
